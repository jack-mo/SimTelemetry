Development

SimTelemetry is an open source project licensed under the GPLv3 license. Please review the GNU site for the complete details.

SimTelemetry is developed in C# 4.0. The source can be found on GitHub. Any help in development is highly appreciated due to the immense scale of various race simulators out there, and enormous amount of applications SimTelemetry could support. GitHub is a great platform for sharing and collaborating for software projects. You can fork the project, work on your own modifications and perform a pull request so the changes can be made to the main distribution of the SimTelemetry core.

It is also possible to write your own plug-ins for SimTelemetry without touching the core program. On this page you can also find more about API documentation.

 

About the program

SimTelemetry consists of 2 programs:

  • LiveTelemetry.exe This is the main 'telemetry application' users will start and , as the name suggests, displays live telemetry to the user. It also runs a telemetry logger on the background which will record the driving done which can be analyzed later.
  • SimTelemetry.exe The telemetry viewer. This has been seperated out so one can easily open multiple telemetry files simlustanously without creating conflicts in the data recording section.
LiveTelemetry has the possiblity to load plug-ins which can serve as another game data stream, data output or UI widget. These plug-ins rely purely on the SimTelemetry.Objects.dll which contains all definitions and interfaces required for posting or receiving data from/to SimTelemetry.
